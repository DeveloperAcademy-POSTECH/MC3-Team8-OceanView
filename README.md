<h2> 🚨 호텔룸오션뷰 Git Workflow </h2>

1. 우리팀 상황
- 팀원 5명
- 협업툴이 모두 다르며 , 사용이 익숙한 팀원이 다수 

2. 브랜치 정의
- main: 제품으로 출시될 수 있는 브랜치 (사용)
- develop : 다음 출시 버전을 개발하는 브랜치 (사용)
- feature : 기능을 개발하는 브랜치 (사용)
- release : 이번 출시 버전을 준비하는 브랜치 (미사용)
